Control Valve Sizing

Reference: Masoneilan Control Valve Sizing Handbook (typical equations). Enter fluid properties and operating conditions; the tool computes Cv (liquid/vapor), selects flow regime (for liquids), and basic valve opening estimate given a chosen valve Cv. Units are selectable next to each field.

1) Common inputs

Quantity Value Unit Notes
Mass flow rate, W Used with ρ to compute volume flow for liquids
Upstream pressure, P1 Absolute or gauge is supported via Pressure (A/G)
Downstream pressure, P2 Same type as P1; ΔP = P1 − P2 (converted consistently)
Pressure drop ratio factor, Fl (or Cf) dimensionless Typical default ≈ 0.9 when unknown (see Note 3)

2) Liquid service

Quantity Value Unit Result / Notes
Density, ρ Used to compute volumetric flow
Vapor pressure, Pv Absolute pressure at flowing temperature
Critical pressure, Pc Used for liquid critical pressure drop limit
Liquid flow type (computed) — Subcritical if ΔP < Fl² ΔPs
Volumetric flow, V V = W / ρ
ΔP (consistent A/G) ΔP = P1 − P2, converted to same base
ΔPs (subcritical threshold) ΔPs = P1 − (0.96 − 0.28(Pv/Pc)0.5)·Pv
Cv (liquid) – final — Subcritical: Cv = 1e3 · V / √ΔP. Critical: Cv = 1e3 · V / (Fl √ΔPs) (units matched to legacy US customary; see below)
Selected valve Cv — From your lookup or spec sheet
Valve opening (estimate) % Opening ≈ Cv / Cvselected · 100%

3) Vapor/gas service

Quantity Value Unit Result / Notes
Molecular weight, Mw kg/kmol Used for specific gravity vs air
Flowing temperature, T Absolute temperature preferred (K or R)
Specific gravity, Gf — Computed: Gf = Mw / 28.96 (air basis)
Compressibility, Z (optional) — If unknown, assume ~0.9–1.0
ΔP (consistent A/G) ΔP = P1 − P2, converted to same base
Per ISA/legacy correlation, Y factor — Y = min(1.5, 1.63 / Fl · (ΔP/P1)0.5)
Cv (vapor/gas) – final — Approx. legacy form: Cv ≈ 0.0545 · W · √(T · Z / Gf) / (P1 · Y) with consistent units
Selected valve Cv — From your lookup or spec sheet
Valve opening (estimate) % Opening ≈ Cv / Cvselected · 100%

Notes

  1. Liquid Critical Flow means cavitation or flashing.
  2. Butterfly valves are at 90° open.
  3. SSW default value Cf (or Fl) = 0.9.
  4. Both wafer and nozzle type check valves are non‑slam, but wafer is cheaper and less robust. The nozzle type is used where catastrophic failure would damage compressors or other critical equipment.
  5. Coefficient Cv signifies the flow [US gal/min] of water at 60 °F through an open valve at 1 psi differential. Coefficient Kv signifies the flow [m³/h] of water at 15 °C through an open valve at 1 bar differential.
    Relationship: Kv ≈ 0.856 · Cv

This tool provides typical equations and approximations for preliminary sizing. Always check the manufacturer’s sizing handbook and noise/cavitation limits for final selection.
